# Serviço de Pagamentos

Projeto desenvolvido em JavaScript para praticar conceitos de:

- Classes
- Métodos
- Propriedades privadas
- Regras de negócio
- Testes unitários

---

## Funcionalidades

- Realizar pagamentos
- Consultar último pagamento realizado
- Categorizar pagamentos automaticamente:
  - `cara` → valor maior que 100
  - `padrão` → valor menor ou igual a 100

---

## Tecnologias utilizadas

- JavaScript
- Node.js
- Mocha
- Assert

---

## Estrutura do projeto

```text
src/
 └── servicoDePagamento.js

test/
 └── servicoDePagamento.test.js
```

---

## Como executar o projeto

Instale as dependências:

```bash
npm init -y
npm i mocha
```
```bash
npx i mocha
```

## Como executar os testes

```bash
npx mocha
```
